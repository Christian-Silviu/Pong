Remake of Pong using C# and MonoGame.

P1 - Left Side
Controls:
W = Up
S = Down

P2 - Right Side
Controls:
Up Arrow = Up
Down Arrow = Down

The win condition for either side is to reach 7 points, with both sides' scores visible in the top center of the screen.

Upon either P1 or P2 winning, press "Enter" to restart the game with a fresh score.
